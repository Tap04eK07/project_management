Description
  ​The Student Project Tracker is a console-based application written in Python using Object-Oriented Programming (OOP) principles. It is designed to help students and educators manage academic projects, assign tasks to specific users, track progress, and generate execution reports. All data is persistently stored in a JSON format, allowing users to save and load their progress at any time.
  
​Requirements:
  ​To run this project, you need the following:
  ​Python Version: Python 3.6 or higher.
​  Libraries: No external dependencies are required (uses built-in json and os modules).
  ​Operating System: Windows, macOS, or Linux.

Installation & Setup:
  1.Download the code: Ensure the file project_manager. py is saved on your computer.
  2. Open Terminal/Command Prompt: Navigate to the folder containing the file.
  3. Verify Python installation: python --version

Usage Instruction:
 Run the application using the following command:python project_manager.py

Main Menu Options:
  1-2: Manage Projects (Create or Delete).
  3-5: Manage Tasks (Add tasks to projects, Assign to students, Update status).
  6: View Report (See a summary of all projects and task completion).
  7-8: Data Persistence (Save to data. json or Load from it).
  O: Exit the program.

Program Logic Structure:
​  The application follows a hierarchical structure to ensure data integrity and clear organization.
